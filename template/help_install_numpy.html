下载校服务器平台的numpy一键安装包

方法一：

下载共享文件scienv.tar.gz，将它拷贝并解压在自己的用户目录下。运行：

$source sci/bin/activate

这样，你就开启了python的虚拟环境。

提交作业

值得注意的是，在提交作业的时候，服务器的环境并没有被配置好，所以需要自己建立一个bash脚本先让服务器初始化环境再提交，案例如下：

#!/bin/bash
source ~/.bashrc #to initialize the environment to run
source ~/sci/bin/activate #my virtualenv
python ~/Giggle/meanfield/model_bc3.py #my python script

LegBuilder Society

LegBuider is a ideology for physicists which is originated from NanJing University, China.

  1. We are social animals.
  2. DRY(don't repeat yourself) is the key code for cooperation, which reminds the physicists that "Don't reinvent wheels, help to improve the human knowledge."
  3. We are also aimed at building up a bridge between "High Tech" and "Common Knowledge", attract investments/donation for physicist.
